# Silk Visualizer

Beat-synchronized silk animations from any audio.

## Usage

```bash
python silk_visualizer.py <audio> [color] [resolution]
```

## Examples

```bash
# Basic
python silk_visualizer.py song.mp3

# With color
python silk_visualizer.py Theme.mp3 purple

# Max quality 4K
python silk_visualizer.py audio2.mp3 blue 4k
```

## Colors

red, blue, purple, green, gold, cyan, white, pink

## Resolutions

720p, 1080p, 2k, 4k, max
